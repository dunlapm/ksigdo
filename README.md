KsigDo
======

Updated version of KsigDo that works with the 1.0 release of SignalR. Original is from http://ksigdo.codeplex.com/ but was not updated for the release version of SignalR. This example corrects this. 

It is built with:
 * .NET 4.0
 * SignalR 1.0.1
 * jQuery 1.9.1
 * Knockout.js 2.2.1
